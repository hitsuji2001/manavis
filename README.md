## Database

* Income
  - What it will store
    - Date
    - Type (Cash, Bank)
    - Income
    - Category
    - Note

* Expense
  - What it will store
    - Date
    - Type (Cash, Bank)
    - Expense
    - Category
    - Note

* Exercise
  - What it will store
    - Date
    - Activity
    - Category
    - Note
